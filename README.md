# lora-webui

Gradio WebUI for LoRA Training

## Features

* Tuning parameters
* Automatically converting to `Stable Diffusion WebUI` compatible `.safetensors` file

## Installation

* Prepare `venv`

  ```shell
  python3 -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
  ```

* Start the app

  ```shell
  python app.py
  ```
